### Документація по розробленому веб-застосунку

Застосунок призначений для створення та відслідковування завдань. Користувач може створити завдання вписаши його назву, та помітити його як виконане за допомогою його id.

### Документація по розгортанню

#### В цій роботі використовувався Arch Linux 2026.05.01.

Образ можна завантажити тут https://archlinux.org/download/  
Там є декілька шляхів завантаження, обирайте найзручніший.  
При встановленні ОС скористайтесь цим гайдом https://wiki.archlinux.org/title/Installation_guide

#### Рекомендовані вимоги до ресурсів віртуальної машини

- Software: Oracle VirtualBox
- RAM: 4 GB (4096 MB)
- Number of CPUs: 8
- Video Memory: 32 MB
- Storage: 50 GB

\*Рекомендовані в сенсі, ті що використовались мною. Я впевнена, що можна й нижчі, але такі вимоги це нормальний компроміс для швидкої роботи ВМ і не занадто великої затрати ресурсів [основної машини](docs/my-pc-specs.txt).

Спеціальних вимог при встановленні ОС немає, хіба що забезпечити собі доступ до інтернету, бо навідміну від інших популярних дистро це робиться не автоматично.

Під час встановлення ОС, вам треба буде створити користувача та встановити root пароль. Встановлюйте там, що вам заманеться, той користувач, що ви створите, заблокується після запуску скрипта. Все що тут можу сказати, що у користувача student пароль за замовчуванням "123" без лапок, без необхідності його змінювати, бо це не було вказано у вимогах роботи.

Щоб запустити автоматизацію розгортання необхідно спочатку завантажити git командою

`sudo pacman -S git`

Після чого клонувати цей репозиторій

`git clone https://github.com/tamanegich/devops-lab1`
